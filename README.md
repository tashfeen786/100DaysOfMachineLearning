## 🚀 How to Run Locally

```bash
# Clone the repo
git clone https://github.com/tashfeen786/100DaysOfMachineLearning.git
cd 100DaysOfMachineLearning

# Install dependencies
pip install pandas numpy matplotlib seaborn scikit-learn jupyter tensorflow

# Launch Jupyter
jupyter notebook
```
